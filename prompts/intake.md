# 基础信息录入脚本

## 开场白

```
我来帮你创建一个属于自己的 Skill。

这是一个把你自己蒸馏成 AI 的过程——我会问你 4 个问题，
然后你可以选择性提供一些原材料（聊天记录、日记、照片等）。

准备好了吗？
```

---

## 问题序列

### Q1：slug（必填，仅限 a-z0-9_）

```
先定一个 slug（命令触发词），只允许小写英文字母、数字、下划线。

比如：xiaobei / me_2026 / my_self
```

**验证**：必须匹配 `^[a-z0-9_]+$`。

---

### Q2：显示名/昵称（必填，可中文）

```
再给这个自己起一个显示名（可中文昵称）。

比如：小北 / 自己 / 20岁的我 / 北某人
```

---

### Q3：基本信息（可跳过）

```
用一句话介绍一下自己？想到什么说什么。

可以包括：
- 年龄 / 年龄段
- 职业 / 身份
- 所在城市
- 教育背景

示例：
  "25 岁，互联网产品经理，上海"
  "研二学生，计算机专业，北京"
  "自由职业插画师，大理"

跳过也没关系，直接回车。
```

**解析字段**：
- `age`：年龄/年龄段
- `occupation`：职业/身份
- `city`：城市
- `education`：教育背景

---

### Q4：自我画像（可跳过）

```
最后一个：用一句话描述你自己？

可以包括：
- MBTI / 星座
- 性格特点
- 别人（或你自己）对你的典型评价
- 你觉得自己最特别的地方

示例：
  "INTJ 摩羯座 社恐但话痨 深夜emo型选手"
  "ENFP 天秤座 社交蝴蝶 但经常感到空虚"
  "不知道MBTI，但我很拖延，对感兴趣的事又会 obsess"

跳过也行。
```

**解析字段**：
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `impression`：主观印象/自我评价

---

## 汇总确认

```
好的，我整理一下：

  slug：{slug}
  显示名：{name}
  基本信息：{summary}
  自我画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
